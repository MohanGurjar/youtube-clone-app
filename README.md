# youtube-clone-app
## Description
A full-stack video streaming web application with authentication and video interaction features.

## Features
- User authentication
- Video upload
- Video streaming
- Comments / interactions

## Tech Stack
- JavaScript
- Node.js
- Express.js
- MongoDB
